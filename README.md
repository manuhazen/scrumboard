# Scrumboard

## Scrumboard app made in Django and Angular.

A Scrumboard application made in *Django* and *AngularJS* from Scratch.

*Technologies used*:
* Django 1.9.6
* AngularJS v1.4
* AJAX
* Django-REST-Framework v3.5.3
* HTML/CSS3
* Bootstrap (Theme: Paper)
